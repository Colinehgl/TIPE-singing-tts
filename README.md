# TIPE : Voix de synthèse pour chant numérique


Réalisation d'un logiciel Text-To-Speech (synthétiseur vocal) capable de chanter.dans le cadre du TIPE.

L'objectif de ce projet est de réaliser un logiciel permettant à l'utilisateur, muni d'un texte et d'une mélodie simple de générer un fichier wav qui contiendrai un chant correspondant au paroles, avec le texte proposé.

---

Le dossier frontend concentre les fichiers en lien avec l'interface utilisateur.
le dossier process concentre le code en lien avec la formatisation du texte donnée en entrée par l'utilisateur.
le dossier singing concentre le code en lien avec l'utilisation du TTS lié au chant.
le dossier tts concentre le code même du synthétiseur vocal.

l'interface utilisateur n'étant pas achevée, il n'y a pas encore de fichiers en dehors du code source pour lancer le logiciel.
